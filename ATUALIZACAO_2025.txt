### Descobertas:
- **Botao:** `button.jobs-apply-button[data-live-test-job-apply-button]`
- **Problema:** Muitas vagas sao externas (gerenciadas fora do LinkedIn)
- **Solucao:** Detectar texto "Respostas gerenciadas fora do LinkedIn" e salvar em arquivo

### Seletores Atualizados:
```python

    "button.jobs-apply-button[data-live-test-job-apply-button]",
    "button.jobs-apply-button",
    "button[aria-label*='Easy Apply']",
    "button[aria-label*='Candidatura simplificada']"
]
```

### Arquivo Gerado:
- `vagas_externas_linkedin.txt` - Vagas para candidatura manual

## CATHO

### Descobertas:
- **Botao:** Botao rosa "Quero me candidatar"
- **Classe:** `Button__StyledButton-sc-1ovnfsw-1`
- **Atributo:** `title="Quero me candidatar para vaga de..."`

```python
seletores_candidatar = [
```

## INFOJOBS

### Descobertas:
- **Botao:** Link `<a>` com texto "CANDIDATAR-ME"
- **Classe unica:** `js_btApplyVacancy`
- **Tipo:** Link com JavaScript, nao button

### Seletores Atualizados:
```python
seletores_candidatar = [
```

## INDEED

### Descobertas:
- **Botao:** Button com ID unico `indeedApplyButton`
- **Atributo de teste:** `data-testid="indeedApplyButton-test"`
- **Mais confiavel:** ID unico raramente muda

### Seletores Atualizados:
```python
seletores_candidatar = [
    "button#indeedApplyButton",
    "button[data-testid='indeedApplyButton-test']",
    "button.indeed-apply-button",
    "button:has-text('Candidatar-se')"
]
```
